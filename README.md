# IntelliJ Icarus Plugin

<!-- Plugin description -->
A development toolkit, providing integration of Icarus build system into JetBrains' products.
<!-- Plugin description end -->


## Install from JetBrains Marketplace
[Download from JetBrains Marketplace](https://plugins.jetbrains.com/plugin/30610-icarus)


## Build
Run `make build-plugin`.


## Install from disk
Open `Settings/Preferences -> Plugins`, choose `Install Plugin from Disk...`, select the ZIP from `build/distributions/`, then restart IntelliJ IDEA.
