# IntelliJ Icarus Plugin

<!-- Plugin description -->
Icarus is an IntelliJ plugin that interfaces with the Icarus Builder build system.
<!-- Plugin description end -->

It adds the `Icarus` dropdown menu in the menu bar.


## Build
Run `./gradlew clean buildPlugin`.


## Install from disk
Open `Settings/Preferences -> Plugins`, choose `Install Plugin from Disk...`, select the ZIP from `build/distributions/`, then restart IntelliJ IDEA.
